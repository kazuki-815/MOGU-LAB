ぽんLINEファネル — 媒体別の可視化

横軸は1月・2月・3月のまとまりで、その中にInstagram / YouTube / stand.fm / noteの4本が並びます（グループ化した縦棒）。 縦軸は流入数（人）です（③の面談予約は件数）。データは funnel_raw.csv と同じです。

流れの図解（概念）
各媒体（Instagram / YouTube / stand.fm / note）から ぽん通常LINEに人が入る → その後 ぽんイベントLINEに移行 → イベントLINE側の導線で 個別（面談予約）に至った件数が、下段グラフの月ごとの4媒体の棒で見えます。

① ぽん通常LINE — 各媒体からの流入（人数）

② ぽんイベントLINE — 各媒体からの流入（人数）

③ 個別（面談予約）— 媒体別の件数

数値は funnel_raw.csv を編集したうえで、build_dashboard.ps1 を実行すると LINE_funnel_dashboard.xlsx と funnel_chart_data.js の両方が更新されます（この HTML は funnel_chart_data.js を読み込みます）。

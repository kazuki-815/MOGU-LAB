## 2. 全LPに共通する「鉄板パターン」

競合4社が全て採用している要素 = ユーザーが期待する「業界標準」であり、外すとCV低下リスク大。

### 2-1. 構成上の共通項

| # | 共通要素 | 詳細 |
|---|---------|------|
| ① | **動画を最上部に配置** | FV直下またはFV内に動画埋め込み（VSL・対談・サンプル）。文字だけでは読まれない前提。 |
| ② | **無料で何かを"あげる"CTA** | 無料講座・無料プレゼント・無料セミナー・無料相談。「無料」が前提のオファー。 |
| ③ | **実績＝ビフォーアフター的数値** | 「-4kg」「-22cm」など具体数値 + 免責注記のセット。 |
| ④ | **LINE公式への誘導** | 全社が最終的にLINE友だち追加 or LINEからの予約に誘導。 |
| ⑤ | **痩せない"原因"の教育パート** | 「あなたが悪いのではない、方法が間違っていた」という免罪符フレーム。 |
| ⑥ | **法的表記セット** | プライバシーポリシー・特商法はフッターに必須。 |

### 2-2. ライティング上の共通パターン

| パターン | 例文（各社共通） |
|---------|----------------|
| **敵の特定（誰が悪か）** | 「情報が悪い」「間違った方法が悪い」→ ユーザーは悪くない |
| **共感ファースト** | 「糖質制限したけど戻った」「何度も挑戦してきた」 |
| **理想の未来を描く** | 体重数値・見た目・髪・肌・メンタルなど複合的未来像 |
| **緊急性・希少性** | 「先着〇名」「残り〇席」「ページを閉じると見られなくなる」 |
| **勧誘しない宣言** | 「無理な勧誘は一切ありません」 |

## 3. 各LPの独自優位性（差別化ポイント）

### LP①：さこダイエット
**「対談動画」という形式の親近感設計**
- 1人での説明ではなく"対話形式"でコンテンツを見せる。視聴者が会話に参加しているような没入感。
- 複数本の動画（対談①②など）でシリーズ性を出し、コンテンツを分割配布することで LINEの開封率・フォロー維持に機能。
- キャッチ「あなたが痩せない本当の理由と今すぐできるコツ」→ 好奇心を煽る"理由"訴求型タイトル。

### LP②：けんとのダイエット講座（STARLIGHT FITNESS）
**「免責注記の多さ」が逆説的な信頼性証明**
- 4ページ以上に渡って「必ずしも全員がそのような結果になるとは限りません」と繰り返し記載。
- これは"実績を大量に掲載している証拠"であり、証拠の量がそのままLPの厚みになっている。
- 男性発信者（けんと）が女性向けダイエットを教える → クロスジェンダー権威型（珍しいアプローチ）。
- 「LINE1日目LP用サンプル動画2分」→ 短時間動画でハードルを下げ、続きをLINEで渡す"チラ見せ戦略"。

### LP③：SAYAKA（アラフォー女性向け）
**圧倒的な権威性 × ターゲット特化の組み合わせ**
- 出版実績（プレジデント社）、Amazonダイエット部門1位（MEGUMIさんを抑えて）、地上波TV、自治体・小学校からの依頼、自スタジオ開業。
- 「アラフォー女性」という年代特化で競合差別化。ターゲットが「自分のための情報だ」と感じやすい。
- 独自理論「骨格と腸にアプローチ」→ 一般的な「糖質制限・カロリー制限」への対立軸を明確に設定。
- 57分の長尺動画を提供 → 長い＝価値が高いという認知戦略。閲覧中に信頼を深める。
- シークレットセミナー（「アラフォーが美しく痩せるたった1つの方法」）→ 限定感 × 希少性。
- 「1分後に公式LINEに届く」という緊急性のある誘導文。

### LP④：ひとみ先生（個別相談会LP）
**「個別相談会」という1対1パーソナル感の訴求**
- 「相談会」という言葉でコンサルや入会ではなく"相談"ハードルに下げ、参加障壁を最小化。
- 特典動画をFV近くに配置し、動画→CTA流れをシンプルに圧縮。
- ©2025 hitomi という"ブランド名のみ"のシンプルなフッター → 個人ブランドとして親近感設計。

## 4. もぐラボとの比較（現状LPの強み・弱み）

### 現状の強み
| 強み | 評価 |
|------|------|
| 14セクションの充実した情報量 | ◎ 業界標準よりも手厚い |
| 4タイプ分類という独自メソッド | ◎ 他社にない差別化軸 |
| 実績数値（Case01〜07） | ◯ 数値＋ケース数で説得力あり |
| CTAの多さ（6箇所+固定） | ◯ 離脱ポイントをカバー |
| ブランドデザインの洗練度 | ◯ 明朝体・ゴールド・女性向けとして上質 |

### 現状の弱み（競合と比べた不足点）
| 弱み | 補足 |
|------|------|
| **VSL動画が未設定** | 全競合が動画を核にしている中、もぐラボはまだ動画なし（最重要課題） |
| **権威性が弱い** | 講師の名前・写真・実績が未確定。SAYAKAのような"圧倒的権威"ブロックがない |
| **ターゲット絞り込みが甘い** | 「20〜40代女性」では広すぎる。SAYAKAのような年代特化があるとCVが上がりやすい |
| **教育パートが長すぎる可能性** | 4タイプ各タイプに長い説明 → 読まれない可能性。チラ見せ→カウンセリングで解説の流れに変える余地あり |
| **"対話"感の不足** | 対談形式や動画内でのQ&A形式など、一方的な情報提供ではなく会話感がない |

## 7. もぐラボ固有の差別化軸（競合にない強み）

競合との比較を通じて、もぐラボが主張すべき独自ポジションを整理する。

| 軸 | もぐラボの主張 | 競合の状況 |
|---|--------------|-----------|
| **タイプ分類メソッド** | 4タイプ診断 → 個別アプローチ | 「アラフォー特化」「骨格+腸」などの独自軸はあるが、複数タイプ分類は少ない |
| **コミュニティ型サポート** | コンサルコミュニティ（仲間と伴走） | 1対1型が多い。コミュニティの安心感・継続率はもぐラボの独自強み |
| **「食べ痩せ」哲学** | 我慢ゼロ・食べながら痩せる | 競合も類似メッセージはあるが、コピーの深掘りで差別化可能 |
| **講師2名体制** | 食べ痩せ × 習慣化の専門家2名 | 単独講師が多い。2名体制は「多角的サポート」として訴求できる |

## 8. 参考：CVR向上のためのコピーライティング原則（競合から帰納）

競合4社のコピーから導き出した"刺さる言葉の型"。

### ヘッドライン型
- **「理由型」**: 「あなたが痩せない本当の理由」（さこ）
- **「年代特化型」**: 「20代の頃のような私を諦めたくないアラフォー女性へ」（SAYAKA）
- **「無料プレゼント型」**: 「ダイエット講座を無料プレゼント」（けんと）

### 共感→解決の型
```
[痛みの描写] → [あなたは悪くない宣言] → [本当の原因] → [解決策の予告] → [証拠] → [行動喚起]
```

### CTA文言の型（全社共通）
- 「まずは話を聞いてみる」（低ハードル言語）
- 「無料で〇〇する」（価値の明示）
- 「先着〇名」（希少性・緊急性）
- 「無理な勧誘は一切ありません」（安心の担保）

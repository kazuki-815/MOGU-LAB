# 特典①スライド用画像

## いちばん簡単な見方（Marp なし）

同じフォルダの **`slides-viewer.html`** をダブルクリックし、**Edge または Chrome** で開いてください。  
「前へ／次へ」またはキーボードの **← →** でスライドが切り替わります。

---

`tokuten1-oyatsu30.marp.md` 側では、各スライドは `![](images/placeholder.svg)` または **PNG / JPEG** を参照しています。

## ファイル名ルール

| スライド上の番号 | 保存ファイル名 |
|------------------|------------------|
| 1 | `item-01.png`（または `.jpg`） |
| … | … |
| 30 | `item-30.png` |

Marp の参照は `.png` になっています。JPEG にする場合は `tokuten1-oyatsu30.marp.md` 内の該当行を `.jpg` に置き換えてください。

## 撮影のコツ

- パッケージ正面が読める明るさ・ピント
- 背景は白〜薄いグレーだとスライドと馴染みやすいです

プレビュー用に、未配置の番号でも `placeholder.svg` をコピーして `item-XX.png` として使うとレイアウト確認がしやすいです。
